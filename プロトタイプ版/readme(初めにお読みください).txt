<ゲームの終わり方>
・escキーで終了します
・又、タイトルの「ゲームをやめる」でも終了出来ます

<コントローラー>
・GamePad推奨 DInputです
・キーボードでは1Pのみ操作可能です
・マウスも対応しています

<操作方法>
・移動 : Lスティック AWSDキー

・ジャンプ : ABボタン スペースキー

・砲撃 : ZRボタン Bキー 左クリック

・カメラ操作：Rスティック マウスドラッグ


<ゲーム概要>
・協力型PvEとの事で最大4人で遊べるゲームにいつかなります！（サーバー実装した頃ぐらいかな...？）


<プロトタイプ版ゲームの一連の流れ>
1,タイトルからポチポチ(ゆっくりでいいよ)連打してゲーム画面に推移します
2,ゲーム画面でトリガーハッピーせずにゆっくり弾を撃って敵を殲滅する事を君はしてもいいし、しなくてもいい
3,バグらずボスを倒せたらゲームクリア！以降ループです

<現在確認しているバグ、不具合>
・Releaseの状態で弾を連射するとたまにバグる<要注意>
・タイトルウインドウで素早く連打するとウインドウが引っ込むだけ
